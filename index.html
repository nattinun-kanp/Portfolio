Predicting Fixed Deposit Account Openings Using Data Mining and Machine Learning

This research project presented a predictive model using data mining techniques to prepare and refine the dataset. After preprocessing, several machine learning algorithms were applied, including:

  • Logistic Regression
  • Decision Tree
  • Bernoulli Naive Bayes
  • Linear SVM
  • XGBoost (Extreme Gradient Boosting)

The analysis focused on predicting customer fixed deposit account openings, based on marketing campaign data from a financial institution. The dataset used was TMCP-02N, containing 32,951 samples.

Results

From the experiments, XGBoost achieved the best performance, with an accuracy of 0.91 (91%), and showed a good balance between precision and recall. This demonstrates the model's reliability in predicting customer decisions.

Future Work

To enhance model performance, additional customer behavioral data such as contact duration and frequency should be included. Moreover, more advanced techniques such as Deep Learning or Cost-sensitive Learning can be explored to handle imbalanced data. Finally, deploying the model in real-world scenarios would help evaluate its practical effectiveness.
