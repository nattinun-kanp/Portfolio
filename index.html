Smart Aquarium Project

This project involves the development of a smart aquarium system using an ESP32 microcontroller. The system features remote fish feeding via a mobile application, real-time monitoring of water temperature, pH, and waste levels, and automatic temperature regulation.

Ride 'N' Guide

Ride 'N' Guide is a personal travel planning assistant powered by AI, designed to help tourists plan their trips efficiently. It provides accurate, tailored recommendations such as attractions, convenient routes, weather updates, and cost estimates—reducing research time and making travel easier and more enjoyable. The system integrates external APIs like Google Maps and OpenWeather to ensure up-to-date and reliable information. Users can explore destinations, check optimal routes, and view weather conditions all in one place, ensuring smooth and personalized travel planning.
